# Day 1: Linux User Setup with Non-Interactive Shell

### Task Overview
Provisioned a new user on a remote server and configured appropriate access levels. This involved navigating cross-machine communication and managing local system permissions. 

### Tools Used
* **SSH**: for secure remote terminal access
* **User Utilities**: 'useradd' command
* **Permissions**: -s shell flag to ensure non-interactive shell

### Key Command Notes
sudo useradd -s /bin/false username

/bin/false chosen as shell flag to simply not allow an interactive shell
