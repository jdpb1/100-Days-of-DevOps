# Day 2: Temporary User Setup with Expiry

### Task Overview
Provisioned a new user on a remote server and configured the account to expire by a predetermined date 

### Tools Used
* **SSH**: for secure remote terminal access
* **User Utilities**: 'useradd' command
* **Permissions**: -e flag to set expiration date

### Key Command Notes
sudo useradd -e YYYY-MM-DD username
